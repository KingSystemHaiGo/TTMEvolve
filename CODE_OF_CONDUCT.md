# 行为准则 / Code of Conduct

TTMEvolve 希望成为一个务实、尊重彼此、适合讨论 AI 辅助游戏开发工具的项目空间。

TTMEvolve aims to be a practical, respectful place for people building AI-assisted game development tools.

## 期望行为 / Expected Behavior

- 尊重他人，给出建设性反馈。
- 默认善意，但在被指出问题时快速修正。
- 讨论尽量围绕项目、用户和可验证事实。
- 报 bug 时提供清晰复现步骤；做设计反馈时给出具体理由。

- Be respectful and constructive.
- Assume good intent, but accept correction quickly.
- Keep discussions focused on the project, users, and verifiable facts.
- Give clear reproduction steps for bugs and concrete reasoning for design feedback.

## 不可接受行为 / Unacceptable Behavior

- 骚扰、威胁、人身攻击。
- 发布私有凭据、认证状态或用户数据。
- 故意夸大或误导 release、安全、能力边界。
- 垃圾信息或无关推广。

- Harassment, threats, or personal attacks.
- Publishing private credentials, auth state, or user data.
- Deliberately misleading release, safety, or capability claims.
- Spam or unrelated promotion.

## 报告 / Reporting

安全或行为相关问题请参考 [SECURITY.md](SECURITY.md) 的联系方式。维护者可以删除评论、关闭 issue，或屏蔽参与者，以维护项目健康。

For security-sensitive or conduct-sensitive reports, use the contact path in [SECURITY.md](SECURITY.md). Maintainers may remove comments, close issues, or block participants to keep the project healthy.
