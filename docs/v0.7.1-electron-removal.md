## 4. 执行流程（必须用户授权）

### 4.1 备份

```bash
python scripts/remove-electron.py --apply --backup
```

这会执行：
1. 把 `electron/` 移动到 `.archive/electron-v0.6.x/`（保险）
2. 删除 `start-gui.*` / `start-practice.*` / `TTMEvolve-Practice.vbs` / `TTMEvolve Practice.lnk`
3. 报告删除的文件清单和总大小

### 4.2 验证

```bash
# 1. dry-run 检查
python scripts/remove-electron.py

# 2. 测试套件全跑
python -m pytest tests/ -q

# 3. 启动 Tauri 验证
./start-tauri.sh    # Linux/macOS
start-tauri.bat     # Windows
```

### 4.3 回滚（如果出问题）

```bash
# 从 .archive/electron-v0.6.x/ 恢复
mv .archive/electron-v0.6.x electron

# 恢复启动器
git checkout start-gui.bat start-gui.ps1 start-practice.bat start-practice.ps1
```
