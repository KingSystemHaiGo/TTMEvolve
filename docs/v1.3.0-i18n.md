## 1. 架构

```
core/i18n.py — Python 翻译函数
i18n/
├── en-US.json — 英文翻译（默认）
└── zh-CN.json — 中文翻译（fallback chain: zh-TW → zh-CN）

frontend/src/lib/i18n.ts — 前端等效包装（未来 v1.4.0）
```

## 2. 使用

### 2.1 Python 端

```python
from core.i18n import t, set_locale, get_locale

# 翻译当前 locale 的字符串
print(t("app.name"))  # "TTMEvolve"

# 切换 locale
set_locale("zh-CN")
print(t("settings.title"))  # "设置"

# 显式覆盖 locale
print(t("settings.title", locale="en-US"))  # "Settings"

# 占位符替换
print(t("app.version", version="1.3.0"))  # "版本 1.3.0"

# 复数
from core.i18n import pluralize
print(pluralize("tools.count_one", "tools.count", 5))  # "5 个工具"
```

### 2.2 前端（v1.4.0 计划）

```typescript
import { t, useLocale } from '@/lib/i18n';
const [locale, setLocale] = useLocale();
return <h1>{t('settings.title')}</h1>;
```

## 3. Locale 解析顺序

1. `t(key, locale="zh-CN")` 显式参数
2. `set_locale()` 设置的活动 locale
3. `TTM_LOCALE` 环境变量
4. `DEFAULT_LOCALE`（en-US）

前缀 fallback：`zh-TW` → `zh-CN`
