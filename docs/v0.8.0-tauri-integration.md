# v0.8.0 — Tauri 集成 + 自动桥接

> 状态：**完成 + 276/276 测试通过 + tag v0.8.0**
> 目标：Tauri 启动时自动拉起 fast_ops HTTP 桥接，统一生命周期管理

---

## 1. 启动流程（v0.8.0）

```
Tauri Rust 主进程启动
    │
    ├─ 1. 启动 fast_ops HTTP 桥接（8766 端口）
    │     - 让 Python 后端能调用 Rust 热路径
    │
    ├─ 2. 启动 Python 后端子进程（8765 端口）
    │     - portable/.venv/python.exe main.py --embedded
    │
    ├─ 3. 等待 Python /health 返回 200
    │
    ├─ 4. 加载 WebView2 显示 frontend/dist/index.html
    │
    └─ 5. frontend 通过 tauri.invoke 调用 Rust commands
```

## 2. 关闭流程

```
窗口关闭事件
    │
    ├─ 停止 Python 后端（ServerManager::stop）
    │
    ├─ 停止 fast_ops 桥接（BridgeHandle::stop）
    │
    └─ 释放资源
```
