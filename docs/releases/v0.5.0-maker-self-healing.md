# TTMEvolve v0.5.0 — Maker MCP 自愈 + 知识种子扩展

> 发布日期：2026-06-25
> 上一稳定版本：v0.4.2 Onboarding Closure

## 主要更新

### Maker MCP 全通路自愈

| 组件 | 文件 | 说明 |
|------|------|------|
| 故障分类引擎 | `server/maker_faults.py` | 14 种故障类型，7 层分类体系 |
| 独立修复脚本 | `scripts/one_click_fix_maker.py` | 一键检测 + 自动修复 |
| 全通路验证 | `scripts/verify_maker_mcp_full_path.py` | 8 步端到端验证 |
| 诊断端点 | `server/app_server.py` | 新增 `/maker/repair` 端点 |
| 集成测试 | `tests/test_maker_setup.py` | 8 项核心测试 |

### 知识种子扩展

| 来源 | 条目数 | 说明 |
|------|--------|------|
| Maker MCP Guide | 20+ | 初始化、Git 策略、故障恢复、升级 |
| UrhoX Lua 引擎 | 15+ | Core API、UI、物理、相机、gotchas |
| Coding Agent 模式 | 10+ | CPR 循环、搜索操作、持久化 |

- `learning/knowledge_seeds.py` — 45+ 条知识种子（原 10 条）
- `agent/agent.py` — 集成 `seed_knowledge_base()`

### 指数退避重试

- `core/repair.py` — 新增 `FAULT_MAX_RETRIES`、`_compute_backoff()`、jitter
- 故障恢复策略更健壮

### 技能生成自动导出

- `learning/skill_generator.py` — 技能生成后自动触发跨生态导出
