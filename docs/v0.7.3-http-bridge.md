# v0.7.3 — Rust ↔ Python HTTP 桥接

> 状态：**完成 + 测试通过 + 文档完整，待 Rust 编译验证**
> 目标：让 Python 后端通过 HTTP 调用 Rust 热路径函数（端口扫描 / 日志尾部 / 目录大小等）

---

## 1. 架构

```
┌────────────────────────────────────────────┐
│  Python 后端（port 8765）                  │
│  core/fast_ops_client.py                  │
│  - fast_probe_port / fast_tail_log 等     │
│        ↓ HTTP POST /fast_ops/{command}    │
│        ↓ JSON body / JSON response        │
│  ┌──────────────────────────────────────┐ │
│  │  Rust HTTP 桥接（port 8766）        │ │
│  │  src-tauri/src/fast_ops_http.rs      │ │
│  │  - std::net::TcpListener + thread    │ │
│  │  - 7 个 tauri fast_ops 命令路由      │ │
│  └──────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

## 2. 协议

### 2.1 请求

```
POST /fast_ops/fast_probe_port HTTP/1.1
Content-Type: application/json
Content-Length: 78

{"host": "127.0.0.1", "port": 8765, "timeout_ms": 200}
```

### 2.2 响应

```
HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 65

{"host": "127.0.0.1", "port": 8765, "available": true, "latency_ms": 0.5}
```

### 2.3 命令清单

| 命令 | 用途 |
|------|------|
| `fast_probe_port` | 单端口 TCP 探测 |
| `fast_find_available_port` | 端口扫描 |
| `fast_tail_log` | 日志尾部读取 |
| `fast_dir_size` | 目录大小计算 |
| `fast_list_dir` | 单层目录列表 |
| `fast_format_bytes` | 字节格式化（IEC） |
| `/health` | 健康检查 |

## 3. 已交付

### 3.1 Rust 端

- `src-tauri/src/fast_ops_http.rs` — HTTP 桥接服务器
  - `BridgeConfig` / `BridgeHandle` — 配置和生命周期
  - `start_blocking()` / `start_background()` — 启动入口
  - `handle_client()` — HTTP 请求处理（method / path / body）
  - `dispatch()` — 路由到具体 fast_ops 命令
  - 9 个 Rust 单元测试覆盖协议层

### 3.2 Python 端

- `core/fast_ops_client.py` — HTTP 桥接客户端
  - `_runtime_available()` — 探测 8766 端口
  - `_invoke_rust()` — POST JSON + 解析响应
  - 6 个 fast_ops 公开函数自动 fallback 到 Python
  - `runtime_status()` / `reset_runtime_cache()`

### 3.3 测试

- `tests/test_fast_ops_http.py` — 14 个 Python 测试：
  - 桥接在线时的端到端调用（probe / format / dir / list / tail）
  - 桥接不可用时的 fallback
  - 运行时状态探测
  - `_invoke_rust()` 协议级验证
  - Python vs Rust 一致性测试
