# TTMEvolve v0.6.0 模块索引

> 配套 v0.6.0 代码审查。帮助新贡献者快速理解新增模块。

---

## 方向一：Plan First + Todolist

| 模块 | 作用 |
|------|------|
| `core/plan_format.py` | 结构化计划 schema：`empty_plan`、`normalize_plan`、`plan_progress`、`plan_to_context_block`、`update_step_status`（不可变更新） |
| `core/plan_review.py` | 确定性审查：未知工具、循环依赖、破坏性操作、孤儿步骤 |
| `core/plan_prompt.py` | LLM 计划生成 prompt + JSON 解析器（`extract_plan_from_llm_text`） |
| `core/plan_validation.py` | 每步执行后的验证报告 |
| `agent/react_loop.py` | `_run_plan_first_phase()`：先计划、用户审批、再执行 |

**入口（agent 启用）：** `config.json` 设置 `agent.plan_first_enabled: true`。

---

## 方向二：Coding Agent 强化

| 模块 | 作用 |
|------|------|
| `core/conditional_hooks.py` | 基于谓词的 hook 触发（tool/verdict/ok/expr） |
| `core/hooks.py` | 集成 `select_applicable_hooks`（条件 hook） |
| `agent/subagent.py` | `spawn_subagent` + `spawn_subagent_batch`（并行 fan-out） |
| `core/context_compression.py` | Abidingenuity 启发的滚动摘要 + 循环检测 |
| `core/loop_scheduler.py` | `/loop` 周期执行 + stop predicate + jitter |
| `core/control_loop.py` | PID 控制器（Ima 之五）：stable/drift/diverging 检测 |

## 方向三：Maker 游戏策划与文案

| 模块 | 作用 |
|------|------|
| `learning/game_knowledge.py` | 6 种游戏类型（runner/tower/match3/shooter/idle/puzzle） |
| `learning/copy_knowledge.py` | 文案模板、批评、banned_words 常量 |
| `learning/mechanics_knowledge.py` | 5 类机制规则（core_loop/progression/reward/economy/retention） |
| `learning/maker_cases.py` | 6 个案例 + 设计经验提取 |
| `learning/socratic_planner.py` | 苏格拉底式 GDD 引导 |
| `learning/engine_knowledge.py` | UrhoX 引擎子系统（physics/audio/input/network/graphics） |

## 方向四：知识整合

| 模块 | 来源 |
|------|------|
