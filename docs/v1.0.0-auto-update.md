## 1. 架构

```
Tauri Rust 主进程
    │
    ├─ tauri-plugin-updater (启动时检查更新)
    │     ↓ 检查 GitHub Releases
    │     ↓ 用户确认 → 下载 → 安装 → 重启
    │
    └─ updater 模块（TTMEvolve 包装）
          - is_newer_version()  // 语义版本比较
          - summarize_release() // 发布说明摘要
          - percent_complete()  // 进度计算

Python 后端
    │
    └─ core/updater_client.py
          - latest_release()     // GitHub API 查询
          - check_for_update()   // 检查更新
          - release_card()       // UI 卡片渲染
```

## 2. 配置

### 2.1 Tauri 配置

```json
"plugins": {
  "updater": {
    "active": true,
    "dialog": true,
    "endpoints": [
      "https://github.com/KingSystemHaiGo/TTMEvolve/releases/latest/download/{{target}}/{{arch}}/{{current_version}}"
    ],
    "pubkey": "REPLACE_WITH_RELEASE_SIGNING_PUBLIC_KEY",
    "windows": {
      "installMode": "passive"
    }
  }
}
```

### 2.2 权限

```json
"permissions": [
  "core:default",
  "core:window:default",
  ...
  "updater:default"
]
```

### 2.3 Cargo 依赖

```toml
tauri-plugin-updater = "2"
```
