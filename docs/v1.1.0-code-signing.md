## 2. 平台支持

### 2.1 Windows

- 工具：signtool.exe（Windows SDK）
- 证书：.pfx（EV 证书推荐）
- 算法：SHA256 + RFC 3161 timestamp
- 命令：`signtool sign /f cert.pfx /p pw /fd SHA256 /tr ... /td sha256 setup.exe`

### 2.2 macOS

- 工具：codesign + xcrun notarytool + stapler
- 证书：Developer ID Application
- 强化：runtime + timestamp
- Notarization：上传到 Apple，staple ticket 到 bundle

### 2.3 Linux

- 工具：gpg（GPG 签名）
- 密钥：GPG key ID + passphrase
- 格式：armor detached signature（.asc sidecar）
- Manifest：sha256sums.txt

## 3. 使用

### 3.1 Windows

```powershell
$env:TTM_PFX_PASSWORD = "..."
python scripts\build_portable\build_sign_windows.py `
    --pfx certs\ttmevolve.pfx `
    --artifacts-dir src-tauri\target\release\bundle `
    --dry-run      # 验证能找到文件
# 真实签名
python scripts\build_portable\build_sign_windows.py `
    --pfx certs\ttmevolve.pfx `
    --manifest
```

### 3.2 macOS

```bash
export TTM_APPLE_ID="dev@example.com"
export TTM_APPLE_TEAM_ID="ABC1234XYZ"
export TTM_APPLE_APP_PASSWORD="abcd-efgh-ijkl-mnop"
python scripts/build_portable/build_sign_macos.py \
    --identity "Developer ID Application: Acme Inc (ABC1234XYZ)" \
    --artifacts-dir src-tauri/target/release/bundle/macos \
    --dry-run
# 真实签名 + 公证
python scripts/build_portable/build_sign_macos.py \
    --identity "Developer ID Application: Acme Inc (ABC1234XYZ)"
```

### 3.3 Linux

```bash
export TTM_GPG_PASSPHRASE="..."
python scripts/build_portable/build_sign_linux.py \
    --gpg-key ABC12345 \
    --artifacts-dir src-tauri/target/release/bundle \
    --manifest
```

### 3.4 一键编排

```bash
# Dry-run（不执行任何签名）
python scripts/build_portable/build_portable.py sign-windows --pfx cert.pfx --pfx-password pw --dry-run
python scripts/build_portable/build_portable.py sign-macos --identity "..." --dry-run
python scripts/build_portable/build_portable.py sign-linux --gpg-key ABC12345 --gpg-passphrase pw --dry-run
python scripts/build_portable/build_portable.py publish --repo KingSystemHaiGo/TTMEvolve --tag v1.1.0

# 真实执行（CI 环境）
python scripts/build_portable/build_portable.py publish --repo KingSystemHaiGo/TTMEvolve --tag v1.1.0 --apply
```
