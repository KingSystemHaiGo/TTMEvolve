## 1. 三平台 Tauri 打包

```json
"bundle": {
  "targets": {
    "windows": ["nsis", "msi"],
    "linux": ["deb", "appimage"],
    "macos": ["dmg", "app"]
  }
}
```

| 平台 | 打包格式 | 安装方式 |
|------|---------|---------|
| Windows | NSIS + MSI | 双击安装包 |
| Linux | DEB + AppImage | apt / 直接运行 |
| macOS | DMG + APP | 拖拽到 Applications |

## 2. 启动器增强

### 2.1 三模式

| 模式 | 用途 | 调用 |
|------|------|------|
| `gui` | 默认 — Tauri 桌面壳 | `./start-tauri.sh` |
| `cli` | CLI 模式 | `./start-tauri.sh --cli` |
| `headless` | 无界面后端 | `./start-tauri.sh --headless` |

### 2.2 平台检测

POSIX 启动器 (`start-tauri.sh`) 通过 `uname` 检测：
- `Linux*` → linux
- `Darwin*` → macos

### 2.3 Python 解析顺序（保持）

1. `./portable/python/bin/python3`（内嵌）
2. `./.venv/bin/python3`（虚拟环境）
3. 系统 PATH 上的 `python3`

### 2.4 友好错误

未找到 Python 时：
- 打印 ERROR 消息
- 提示运行 `scripts/build-portable/build_all.bat` 或 `build_all.py`
