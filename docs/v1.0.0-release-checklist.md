## 4. 发布流程

```bash
# 1. 验证测试
python -m pytest tests/ -q

# 2. 时间戳同步
./scripts/cos-time.sh all

# 3. 提交
git add -A
git commit -m "v1.0.0: production-ready release"

# 4. Tag + Push
git tag -a v1.0.0 -m "v1.0.0: production release"
git push origin main
git push origin v1.0.0

# 5. GitHub Release
gh release create v1.0.0 --title "v1.0.0" --notes-file docs/releases/v1.0.0.md
```
